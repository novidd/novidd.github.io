I. About the project

Creation date: Jun.2023

Team size: Solo

Software used: Figma, Unreal Engine 5.1, Photoshop

Production time: 13 days

II. Theme & Goal(s)

As my first diegetic UI, prototype a main menu in Figma, then implement in UE5 with minial interaction.

The inspiration for this UI is the Half-Life 1 horror game mod Cry Of Fear, I managed to find a free model of the exact phone in the game which is modeled after the Sony Ericsson w610i mobile phone.

III. Reflection

What I've learned:

I learned how to make emissive textures to get the glowing buttons.

An important detail to convey the visuals of an older computer screen is to add scanlines, in this case vertical scanlines.
(I also learned how much I don't like building functionality for UI in UE5 ٭(•﹏•)٭)

What I could do better:

All the UI elements could be larger - hard to see.

If you want to make a more readable phone UI it's better to make it an overlay on screen like in for example, Cyberpunk 2077 or Grand Theft Auto. However, for an immersive horror game I think it could still work.
