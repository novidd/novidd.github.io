I. About the project

Team size: Solo

Software used: Figma, Unity, Photoshop

Production time: Figma Prototype (10 days) + Unity (2 days)

II. Theme & Goal(s)

As my first UI, prototyped a main menu in Figma, then implemented it in Unity with at least two internal menus and a simple store page. A goal I set for myself was to mimic the diegetic UI from the terminals in Alien Isolation, eventually with some improvements.

I tried maintaining the Alien franchise identity using Ron Cobb's iconic design work.

For example, this icon means "hazard warning" which I placed on the play game button.
