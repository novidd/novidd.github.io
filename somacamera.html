II. Theme & Goal(s)

The goal with this project was to design a UI based on an already existing IP SOMA using Figma, then implemented it in Unity .

For example, this icon is the SOMA computer operating system pathOS which is used on every terminal screen in the original game. This also also an asset used in the original game, speeding up the process of a polished UI.

Another example, this kanji (chinese-symbol) means "tool".
